A sample command-line application with an entrypoint in `bin/`, library code
in `lib/`, and example unit test in `test/`.

Install locally:
```
dart pub global activate --source=path .
```

Deactivate:
```
dart pub global deactivate dart_analysis
```

Run:
```
dart_analysis .
```
